Panduan Unggah Gambar

Format & Ukuran

  • Format: JPG, PNG, WebP, GIF
  • Ukuran Maksimal: 10 MB per file
  • Resolusi: Minimal 300x300px
  • Resolusi Optimal: 1920x1080px (berita), 1200x1600px (album)

Rasio Aspek

  • Portrait: 3:4 atau 4:5 (untuk album)
  • Landscape: 16:9 atau 4:3
  • Square: 1:1 (untuk thumbnail)
  • Flexible: Sistem akan menyesuaikan

Pemrosesan Otomatis

  • Thumbnail: Dibuat otomatis (300x300px)
  • Optimasi: Kompresi otomatis
  • Visibilitas Default: Unggahan Admin/Editor/Penulis disembunyikan; lainnya terlihat
  • Berbagi Aset Admin: Gambar yang diunggah admin dapat dipakai semua pengguna

Tips Terbaik

Untuk Album:

  • • Gunakan rasio portrait (3:4) untuk konsistensi
  • • Resolusi minimal 800x1200px (optimal 1200x1600px)
  • • Hindari gambar horizontal
  • • Pastikan kualitas tinggi

Untuk Berita:

  • • Gunakan rasio landscape (16:9) atau 4:3
  • • Resolusi minimal 1200x675px (optimal 1920x1080px)
  • • Fokus pada subjek utama
  • • Optimalkan untuk web
